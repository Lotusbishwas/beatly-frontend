# Beatly Frontend

## Overview
Beatly is a modern video sharing platform with a sleek, responsive design built using React and Chakra UI.

## Features
- User Authentication (Signup/Login)
- Video Upload
- Video Browsing
- Like and Comment Functionality
- Responsive Design

## Tech Stack
- React
- Chakra UI
- Axios
- React Router
- React Icons

## Prerequisites
- Node.js (v14+)
- npm or yarn

## Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/beatly.git
cd beatly-frontend
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Set up environment variables
Create a `.env` file with:
```
REACT_APP_API_URL=http://localhost:5000/api
```

## Running the Application
```bash
npm start
# or
yarn start
```

## Building for Production
```bash
npm run build
# or
yarn build
```

## Deployment
- Ensure all environment variables are set
- Use your preferred hosting platform (Vercel, Netlify, etc.)

## API Integration
- Backend API: http://localhost:5000
- Authentication via JWT
- Supports video upload, like, comment, and browsing

## Troubleshooting
- Ensure backend server is running
- Check browser console for any errors
- Verify API endpoints and authentication
